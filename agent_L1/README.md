# 激光切割智能工艺决策系统 (Laser Cutting Copilot)

这是一个集成 **本地专家规则库** 与 **大模型智能体 (LLM Agent)** 的智能激光切割工艺辅助决策系统。系统旨在通过切前视觉检测、智能工艺推荐、切后缺陷视觉反馈，形成“感知-决策-执行-反馈”的闭环，帮助工艺人员快速确定最优的激光切割参数（功率、速度、气压、焦点等）。

---

## 🌟 主要功能

1. **2D/3D 切前视觉检测**
   - 自动生成板材表面模拟特征（如划痕、锈蚀、油污）。
   - 3D 测高扫描与板材翘曲高度分析，输出安全拦截警告。

2. **双工艺决策引擎**
   - **本地专家规则库模式**：基于专家经验数据库，通过材质与厚度快速检索相近工艺，并自动根据厚度进行物理缩放推荐，安全稳定。
   - **大模型智能体 (LLM Agent) 模式**：通过调用本地部署的 Ollama（如 `deepseek-r1:32b`、`qwen2.5-coder` 等）进行深度的缺陷归因与调参推理，输出智能的工艺建议。

3. **闭环试切与缺陷视觉诊断**
   - 模拟真实的切后工件质量评估，计算多维度评分（切口质量、熔渣高度、过烧程度、粗糙度等）。
   - 智能生成工艺调参偏置（Offset），支持一键应用参数修正并重新试切。

4. **专家工艺知识库维护**
   - 工艺数据库的可视化展示与实时材质检索。
   - 支持新增专家工艺并持久化保存。

---

## 🛠️ 技术架构

- **桌面客户端**：基于 `CustomTkinter` 框架开发，提供现代、流畅的深色模式桌面 GUI 界面。
- **Web 服务端**：基于 `FastAPI` (Python) 提供后台 API 服务，前端采用原生极简 CSS+JS 构建，具备高度动效与良好响应性。
- **智能体接口**：支持 OpenAI 格式的 API 接入，专门针对本地 Ollama 框架进行优化（内置防网络代理拦截、超时自动延长等鲁棒机制）。

## 🚀 快速开始

### 1. 环境准备
确保您的计算机上已安装 Python 3.8+，然后安装项目依赖：
```bash
pip install -r requirements.txt
```

### 2. 运行桌面客户端 (GUI)
```bash
python gui_app.py
```
在桌面应用中，您可以通过右侧的 `智能体配置` 面板检测本地 Ollama 运行的模型并进行连接测试。

### 3. 运行 Web 服务与网页端
启动 FastAPI 后台服务：
```bash
python run.py
```
启动后，在浏览器中访问 [http://localhost:8000](http://localhost:8000) 即可打开 Web 主控仪表盘。
